# 变量

## 作用

给一段制定的*内存空间* 起名，方便操作这段内存

## 语法

    数据类型 变量名=初始值

## 示例

    #include<iostream>
    using namespace std;
    int main(){
        
         //变量的定义
         //语法：数据类型 变量名=初始值
        
        int a=10;
        cout<<"a="<<a<<endl;
        system("pause");
        return 0; 
    }

<u>建议：变量名易于理解</u>
